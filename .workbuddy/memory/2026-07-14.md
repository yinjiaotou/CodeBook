### ardot batch_edit DSL 关键坑（本次复确认）
- 多操作必须用**换行**分隔，不能空格连写一行（否则 "Skipped unparseable line" / parse 失败）
- 嵌套建屏：`parent=I("0:1",{...})` 后 `child=I(parent,{...})` 变量串联；`I()` 响应会回传新建节点真实 ID，跨批次引用用真实 ID
- 每批 ≤25 ops；富屏拆 2–3 批
- 截图 `capture_screenshot` 存 `.workbuddy/screenshots/*.webp` 但 **Read 读不到**（ardot 沙箱路径），改以 `capture_layout(parentId, problemsOnly:true)` 做结构校验
- `capture_layout` 参数用 `parentId`（非 nodeIds），且不能传 page 根节点（0:1 是 page，报 "must be a container"）
- 行内分隔线用 strokeBottomWeight 做底部描边，避免插入顺序问题
