# macOS MVP 手工验收清单

> 在隔离的测试资料库中执行；本清单记录可观察行为，不构成绝对安全保证。每项记录日期、macOS 版本和结果。

| 验收项 | 操作 | 预期结果 |
| --- | --- | --- |
| 主密码规则与提示 | 首次启动，输入少于 12 个 Unicode 字符的主密码；再查看创建页。 | 不能创建，显示至少 12 个字符的提示；页面明确说明遗失主密码无法恢复。 |
| 连续解锁失败 | 创建资料库并锁定，连续输入 5 次错误主密码，然后立即输入正确密码。 | 第 5 次后显示等待提示；冷却期间正确密码不解锁；约 30 秒后可再次用正确密码解锁。 |
| 创建、重启、解锁 | 用符合规则的主密码创建资料库，退出应用后重新打开并解锁。 | 创建后进入资料库；重启后显示解锁页；正确密码可恢复访问已有条目。 |
| 条目与筛选 | 新增两个不同分类的条目；编辑其中一项；按标题搜索；切换分类；删除一项。 | 新增、编辑、删除后列表与分类随之更新；搜索和分类共同生效；删除后条目不再显示。 |
| 自动锁定 | 分别设置 3、5、10 分钟，保持无操作至对应时间；每次重新解锁后继续。 | 每种设定都在对应空闲时间后回到解锁页。 |
| 切到后台锁定 | 解锁并选中一个条目，然后让应用进入后台或切换到其他应用。 | 立即回到锁定状态；返回应用时需重新解锁。 |
| 隐藏、显示、复制与计时 | 选中含密码的条目，先确认密码以圆点隐藏，再点“Reveal”、点“Copy”。 | Reveal 前不显示原密码；Reveal 后显示；复制成功后出现“Password copied — clears in 30s”倒计时和“Clear Clipboard”。 |
| 剪贴板替换保护 | 复制密码后，在 30 秒内从其他应用复制一段测试文本；等待倒计时结束，或在 Pwdlock 中点“Clear Clipboard”。 | Pwdlock 的倒计时反馈消失；其他应用后复制的测试文本仍保留，未被 Pwdlock 清空。 |
| 正常剪贴板清理 | 复制密码后不再复制其他内容，观察至约 30 秒，或立即点“Clear Clipboard”。 | 倒计时结束或立即清除后，Pwdlock 的复制反馈消失；可按系统剪贴板查看方式确认原复制项不再可用。 |
| 本地备份与恢复 | 新增条目 A，创建本地备份；新增或修改条目 B；执行“Restore Latest Backup”。 | UI 不显示明文导出入口；恢复后资料库回到备份时的内容（例如 A 存在、备份后改动 B 不在）。 |
| 损坏备份拒绝 | 保留当前可识别条目“保留当前”；将最新备份替换为故意截断/损坏的测试副本后恢复。 | 显示恢复失败；“保留当前”仍可见且可打开，当前资料未被该损坏副本替换。 |
| 无效架构备份拒绝 | 使用预制的“可打开但缺少/破坏登录条目表架构”的测试备份替换最新备份后恢复。 | 显示恢复失败；现有条目和当前解锁会话仍可使用，未接受无效架构备份。 |
| 锁定后的 UI 缓存 | 在详情页 Reveal 密码并复制后，手动锁定；或在后台锁定后返回。 | 解锁前不显示之前的标题、用户名、密码、分类、搜索结果或倒计时反馈；复制操作不可用。 |
| DMG 安装与首次打开（有交付 DMG 时） | 挂载交付的 DMG，将应用拖入“Applications”，再从 Finder 打开。 | 应用可安装并启动；因 ad-hoc 签名，macOS 可能显示未验证开发者/需要额外确认的警告。按组织的 macOS 策略确认，不应把该警告视为安全认证。 |

备注：DMG 打包不属于当前任务；最后一项仅在另行提供 DMG 交付物后执行。
